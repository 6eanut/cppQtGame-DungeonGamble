# cppQtGame-DungeonGamble

一个基于 C++ 和 Qt 实现的简单图形化小游戏，核心玩法是通过选择怪物不断向上爬塔，在风险与收益之间做出决策。

---

项目简介

本项目为《高级语言程序设计》课程大作业，使用 Qt 框架开发图形界面，结合 C++ 实现游戏逻辑。

游戏的基本目标是尽可能爬到更高的层数。玩家在每一层需要从多个怪物中做出选择，不同选择会带来不同结果，需要在有限血量下进行策略判断。

---

基本玩法

* 每一层会出现三个怪物选项
* 只有一个是正确选择，选择后可以进入下一层
* 选择错误会损失血量，但仍可获得金币
* 同一层中，怪物不能重复选择（按钮会变为不可点击）
* 玩家血量归零则游戏结束

---

核心机制

1. 风险决策机制

血量不仅是生命值，同时也是一种“资源”。玩家可以选择稳妥推进，也可以承担风险换取收益。

2. 金币与商店系统

通过战斗获得金币，可以在商店中购买：

* 恢复血量的道具
* 提供提示的道具
* 战斗辅助道具

3. Boss 战

每隔一定层数会进入 Boss 战，采用不同于普通关卡的战斗方式，使游戏节奏更加丰富。

技术实现

* 编程语言：C++
* 图形界面：Qt
* UI设计：Qt Designer
* 样式：QSS（Qt Style Sheet）

主要实现内容包括：

* 多页面切换（QStackedWidget）
* 按钮交互逻辑
* 游戏状态管理（层数 / 血量 / 金币）
* 随机机制与简单概率控制
